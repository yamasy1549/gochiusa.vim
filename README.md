# gochiusa.vim :rabbit:

## Installation

```
# .vimrc

colorscheme gochiusa
```

### dein.vim

```
# dein.toml

[[plugins]]
  repo = 'yamasy1549/gochiusa.vim'
```

### others

```
$ git clone https://github.com/yamasy1549/gochiusa.vim
$ cp gochiusa.vim/colors/gochiusa.vim ~/.vim/colors/
```
